# Environment setpup

Currently using Python 3, if you have mutiple python verisons, you can create a virtual environment:

```
# Create virtual Env
cd NicoJp
virtualenv -p python3 .
```

To cativate the virtual environment, do the usual:
```
cd NicoJp
./bin/activate
```
